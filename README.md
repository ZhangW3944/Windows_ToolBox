# Windows ToolBox 2.7 PLUS

## 📌 项目简介

Windows ToolBox 2.7 PLUS 是一个功能强大的Windows系统工具箱，集成了多种实用功能，包括系统工具、实用程序、维护工具、网络工具等，采用C++开发，支持中英文双语切换。

## ✨ 主要功能

### 🛠️ 系统工具
- 任务管理器
- 注册表编辑器
- 设备管理器
- 磁盘管理
- 服务管理

### 📱 实用工具
- 计算器
- 记事本
- 画图工具
- 截图工具
- 远程桌面

### 🔧 系统维护
- 磁盘清理
- 磁盘碎片整理
- 系统文件检查
- 系统还原
- 恶意软件扫描

### 🌐 网络工具
- IP配置查看
- DNS缓存管理
- 网络连接测试
- 网络故障诊断

### ⚡ 电源管理
- 关机/重启/注销
- 定时关机
- 锁定工作站
- 睡眠模式

### 🔍 高级功能
- 系统信息查看
- 进程管理
- 用户账户管理
- 系统日志查看
- 文件搜索

## 🚀 使用说明

1. **下载**：克隆本仓库或下载release版本
2. **运行**：直接运行`Windows_ToolBox_v2.7_PLUS.exe`
3. **操作**：
   - 使用数字键选择功能
   - 按`L`切换中英文
   - 按`H`查看帮助
   - 按`X`退出程序

## ⚠️ 注意事项

- 部分功能需要管理员权限
- 电源管理操作不可撤销，请谨慎使用
- 修改密码功能仅修改程序内密码，不影响系统账户
